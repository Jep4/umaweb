# 우마무스메 쥬얼 계산기

우마무스메 프리티 더비에서 목표 시점까지 수집 가능한 쥬얼의 양을 계산하는 웹 애플리케이션입니다.


##  기술 스택

### Frontend
- **React** 19.2.0
- **React Hooks**
- **CSS3** 

### Backend
- **FastAPI** 
- **MongoDB** 
- **Motor** 
- **Pydantic**
